# README

The Ruby's version of this repo is 3.0.0 and the Ruby on Rails version is 7.0.4

<h3>Install depedencies</h3>
```
bundle install
```

<h3>Run migrations</h3>
```rails db:migrate```

<h3>Seed the db</h3>
```rails db:seed```

<h3>Run tests</h3>
```bundle exec spec```

<h3>Run server</h3>
```rails s```

the urls are:
```http://localhost:3000/users/:id```
```http://localhost:3000/books/:id```
